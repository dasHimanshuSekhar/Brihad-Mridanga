# **App Name**: BiblioBoost

## Core Features:

- Book Listing & Selection: Display 10 hardcoded books with options for users to select books and quantity.
- Order Placement: Allow users to fill in their details (Name, Shipping Address, Mobile Number, Email) to place an order.
- Referral Input: Provide an option for users to enter a referral code (referrer's Mobile Number).
- Referral Game Display: Visually represent the user's progress toward milestones and rewards in the referral game.
- Public Leaderboard: Show a publicly visible leaderboard of top users ranked by their Mobile Number and total book count.
- Referral Check: Allow any user to enter their Mobile Number to view the total books purchased through their referral code.
- Embedded Game: Integrate a simple, hardcoded game to enhance the user experience.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to evoke a sense of comfort and familiarity, aligning with the joy of reading.
- Background color: Light beige (#FAF9F6), slightly desaturated from the primary color to ensure readability and visual harmony.
- Accent color: Soft brown (#A69074), analogous to the primary, adding a touch of elegance and highlighting key interactive elements.
- Body font: 'Literata', a transitional serif for a literary, vintage, slightly formal feel; appropriate for large amounts of text.
- Headline font: 'Playfair', a modern serif for a more fashionable high-end feel; suited for headers, chapter titles, and short snippets of text.
- Use simple, elegant icons related to books and reading.
- Subtle animations when interacting with the referral game elements.
